# Frontend, Backend et Full-stack dans les frameworks JavaScript

## Objectif de l’exercice

Comprendre le rôle principal des frameworks JavaScript et savoir les classer selon qu’ils sont :

- Frontend
- Backend
- Full-stack

---

## Énoncé de l’exercice

Vous disposez ci-dessous d’une liste de frameworks JavaScript.  
Pour chacun d’eux, indiquez s’il est principalement utilisé pour :

- le **Frontend**
- le **Backend**
- le **Full-stack**
